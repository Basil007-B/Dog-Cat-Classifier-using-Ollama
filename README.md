# 🐾 Dog vs Cat Classifier using Flask & Ollama

AI-powered Dog vs Cat image classifier built with Python, Flask, and Ollama using the LLaVA vision model.  
Runs completely offline on your local machine — no API key required.

---

## 🚀 Features

- 🐶 Detects Dogs and Cats from uploaded images
- 🤖 Uses Ollama + LLaVA for image understanding
- 🌐 Beautiful Flask Web Interface
- ⚡ Real-time streaming AI responses
- 🔒 Fully Offline & Privacy Friendly
- 🎨 Modern responsive UI design
- 📷 Supports JPG, PNG, GIF, WEBP

---

## 🛠️ Technologies Used

- Python
- Flask
- Ollama
- LLaVA Vision Model
- HTML
- CSS
- JavaScript
- Requests Library
